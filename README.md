## Description

Backend service for Noala tech test - serves random images for dogs'n'cats'n'ducks.

Deployed (in a _very_ lazy way) to https://be-dcd.onrender.com/

## Quick start
 - `npm i`
 - `npm run start`


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
